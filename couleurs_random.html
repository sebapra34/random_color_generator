générer une code couleur hexadecimal aléatoire (de type : #FFCC55)

On peut générer un nombre aléatoire en JavaScript avec la méthode Math.rand() mais ce nombre est systématiquement compris entre 0 et 1. Il faut ensuite multiplier le nombre aléatoire généré par la différence entre les deux nombres qui sont les limites et ajouter 1. On obtient ainsi un nombre à virgule. La méthode Math.floor() va récupérer la partie entière de ce nombre. Après ce calcul, il ne reste plus qu'à ajouter le nombre inférieur de l'intervalle pour obtenir un nombre aléatoire compris dans l'intervalle.
